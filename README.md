# PaceBeat

PaceBeat is a run metronome and training companion built to help runners maintain a steady cadence and improve pacing consistency.

## What it does
- Metronome-style cadence guidance during runs
- Run tracking and post-run stats (pace, cadence, time, distance)
- Optional maps/route visualization (when location permission is granted)
- Premium features available via subscription

# Rogue Sudoku: Puzzle RPG

Sudoku evolved into a roguelike puzzle RPG. Chain combos, collect relics, and climb the ranks — every run is different.

## What it does
- Arcade Mode: roguelike runs with shops, relics, and escalating difficulty
- Classic Mode: traditional sudoku (Easy / Medium / Hard)
- Collect Bronze, Silver, and Gold relics that modify gameplay
- Leaderboards, achievements, and progression system
- Free to play with ad support (no pay-to-win)
