# PaceBeat

PaceBeat is a run metronome and training companion built to help runners maintain a steady cadence and improve pacing consistency.

## What it does
- Metronome-style cadence guidance during runs
- Run tracking and post-run stats (pace, cadence, time, distance)
- Optional maps/route visualization (when location permission is granted)
- Premium features available via subscription
